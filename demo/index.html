Quick Installation

Get up and running in under 5 minutes. No complex dependencies required.

1

Clone and Install

Install genbox with the image and video generation packages.

Bash 
# Clone the repository
git clone https://github.com/MarkinHaus/genbox
cd genbox

# Install with all features (recommended)
pip install -e ".[all]"

# Or install selectively
pip install -e ".[image]"  # image generation only
pip install -e ".[image,video]"  # + video generation
2

Run Setup Wizard

The setup wizard detects your hardware and creates an optimized configuration.

Bash 
# Interactive setup - detects VRAM automatically
genbox setup

# Or force re-configuration
genbox setup --force

# Install default models for your VRAM profile
genbox models install-defaults

# Install for specific VRAM profile (dry-run first)
genbox models install-defaults --profile 12gb_balanced --dry-run
3

Start Generating

Use the CLI or launch the web UI for an interactive experience.

Bash 
# Generate an image via CLI
genbox gen -p "a brutalist tower at dusk, fog, cinematic" \
  --model flux2_klein \
  --width 1024 --height 1024

# Generate a video
genbox gen -p "a serene lake at sunrise" \
  --mode t2v \
  --model wan_1_3b \
  --frames 48 --fps 24

# Launch the web UI (Streamlit)
genbox ui  # opens at http://localhost:8501

Python API

Clean, intuitive Python API for seamless integration into your workflow.

T2I Text to Image
Python 
from genbox import pipeline

result = pipeline.text_to_image(
prompt="a cyberpunk cityscape",
model="flux2_klein",
width=1024,
height=1024,
steps=28,
guidance=3.5,
seed=-1,
)
I2I Image to Image
Python 
from genbox import pipeline

result = pipeline.image_to_image(
prompt="transform to oil painting",
input_image="photo.jpg",
model="sdxl_base",
strength=0.75,
)
T2V Text to Video
Python 
from genbox import pipeline

result = pipeline.text_to_video(
prompt="ocean waves at sunset",
model="wan_1_3b",
frames=48,
fps=24,
width=480,
height=480,
)
INP Inpainting
Python 
from genbox import pipeline

result = pipeline.inpaint(
prompt="remove the person",
input_image="photo.png",
mask_image="mask.png",
strength=0.99,
)

MGT Model Management
Python 
from genbox import pipeline

# Check installation
pipeline.is_installed("flux2_klein") # → bool

# Download model
pipeline.download("wan_1_3b", force=False)

# Install defaults for VRAM profile
installed = pipeline.install_defaults(profile="12gb_balanced")

# List models
models = pipeline.list_models(model_type="video")
for m in models:
    print(f"{m.id}: {m.vram_gb}GB")
